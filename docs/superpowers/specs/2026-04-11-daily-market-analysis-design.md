# 每日大盘深度分析设计

*日期: 2026-04-11*

## 背景与动机

单股分析的技术面维度权重已提升到 40%，其中决策树 Q5 要求评估"大盘环境对个股的影响"。但当前大盘数据只有 `macro_intel.py` 提供的 3 行精简摘要（"偏强/偏弱/分化"+ 上证点位），以及 `regime_detector.py` 的四档分类标签。模型在分析个股时缺乏对大盘的深度理解，导致 Q5 节点判断粗糙。

## 设计目标

- 每天第一次单股分析时自动触发一次大盘深度分析（Opus）
- 分析结果按日持久化缓存，当日所有个股分析共享
- 覆盖：四大指数技术面、量能、市场情绪、板块轮动、外围市场（美股/港股）
- 输出一份可直接注入 prompt 的结构化报告

## 架构

```
首次 analyze → 检查当日缓存 → 无缓存 → 数据采集(代码) → Opus 深度分析 → 持久化
                              → 有缓存 → 读取
→ 大盘分析 markdown 注入个股数据包头部 → 个股分析（Round 1 / Round 2）
```

## 触发与缓存

- **触发时机**：当天第一次 `python cli.py analyze` 时自动触发
- **缓存粒度**：按自然日，一天一份
- **存储位置**：`data/knowledge/market_daily/{date}.json`（结构化数据）+ `{date}.md`（格式化文本）
- **缓存检查**：有当日 `.md` 文件就直接读取，否则触发完整分析流程
- **模型**：Claude Opus（MAX）— 每天只跑一次，质量优先

## 数据采集（代码层，不消耗 AI token）

在调用 Opus 之前，先用代码采集全部原始数据：

| 数据 | 来源 | 内容 |
|------|------|------|
| 四大A股指数 | Tushare `get_price_df` | 上证/深证成指/创业板/科创50 近20日行情（OHLCV） |
| 两市总成交额 | akshare | 近10日总成交额趋势 |
| 涨跌家数 | akshare `stock_zh_a_spot_em` | 涨/跌/平家数、涨停/跌停数 |
| 板块轮动 | akshare `stock_board_industry_name_em` | 当日领涨/领跌板块 Top5 |
| 美股三大指数 | akshare | 道琼斯/纳斯达克/标普500 最新收盘+涨跌 |
| 港股恒生指数 | akshare | 恒生指数/恒生科技 最新+涨跌 |
| 当前 regime | `regime_detector.detect_current_regime()` | 牛/熊/震荡/轮动 |

数据采集函数全部包裹 try/except，单项失败不阻塞整体流程。

## AI 深度分析（Opus）

### System Prompt

```
你是一位专业的A股市场技术分析师。请基于以下原始数据，输出一份完整的每日大盘深度分析报告。

要求：
- 所有结论必须基于具体数据，禁止泛泛而谈
- 指数分析必须包含近5日逐日K线点评
- 量能分析必须给出具体成交额数字和趋势判断
- 外围市场必须评估对A股次日的传导影响
- 最终必须给出明确的操作环境评级
```

### 输出格式（7 个区块）

```
### 一、大盘趋势总判
[一句话结论 + 对个股操作的系统性建议]

### 二、四大指数技术分析
[逐一分析每个指数：趋势状态、MA20/MA60位置关系、近5日K线逐日点评（日期+涨跌幅+成交量变化+K线类型）、关键压力位/支撑位]

### 三、量能分析
[两市总成交额趋势（近10日数据）、今日量能水平（放量/缩量/地量）、量价配合度评估]

### 四、市场情绪
[涨跌家数比、涨停/跌停数量、情绪周期判断（冰点/回暖/亢奋/退潮）、情绪持续性评估]

### 五、板块轮动
[领涨板块Top5及上涨逻辑、领跌板块Top5及下跌原因、资金主攻方向判断]

### 六、外围市场
[美股三大指数走势及对A股影响、港股恒生/恒生科技走势及AH联动判断、整体外围环境评估]

### 七、个股操作环境评估
- 大盘环境评级：[强势进攻/偏多震荡/中性/偏空震荡/弱势防御]
- 适合的操作策略：[追强/低吸/观望/减仓]
- 需要警惕的系统性风险：[具体风险或"无"]
- 当前环境下个股分析注意事项：[2-3条具体建议]
```

## 注入个股分析

在 `_run_war_room_v2()` 的 Phase 0 末尾，将大盘分析 markdown 拼入数据包头部：

```python
market_md = get_or_run_market_analysis(analyst_model)
full_data_brief = market_md + "\n\n" + full_data_brief
```

个股分析的 Round 1 在评估技术面 Q5（大盘环境对个股的影响）时，就能直接引用大盘深度分析中的结论和数据。
